# Skripty k článkům

Sem ukládáme skripty, které jsme reálně použili při přípravě článků (sběr dat, čištění, export podkladů pro grafy).

## Konvence

- Název: `generate-<tema>-data.py`
- Skript má být spustitelný přes `uv run --no-project ...`
- Výstupy ukládej do `ai/articles/<tema>/`
- V článku uveď, z jakého skriptu data vznikla

## Aktuální skripty

- `generate-vystavba-data.py` – ČSÚ data k tématu výstavby bytů (povolení vs. dokončené byty)
